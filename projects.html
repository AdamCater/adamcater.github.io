TANK SHOOTER

Tank Shooter challenges the player with different sloped environments which they must traverse and destory enemy tanks to progress the level. The player must find 3 keys to get to the next stage. There is different weapon types such as Sniper Rifles, Flamethrowers and Pistols (and much more).

Solo Project Unreal Engine Blueprints AI Niagara System
MY ROLE

I was responsible for designing and implementing gameplay systems, such as Weapon Spawning and Shooting, as well as integrating assets into the project. AI Behaviours like Wander and Idle was used to enable different AI types. Particle System (Niagara) were also used. I also allowed the functionality of Weapon Swapping, which includes defeating enemies to get better weapon types to defeat other (non-reachable) enemies.

UNKNOWN INVASION

Unknown invasion is a 2D spaceship shooter. The player must defeat enemy spaceships to clear out the invaders trying to destroy planet Earth.

Solo Project SDL C++ UBUNTU EVERCADE
MY ROLE

I was responsible for creating and implementing gameplay systems, such as bullet pooling and shooting, as well as integrating assets into the project. Virtual Machines like UBUNTU was used to enable to port this game onto the EVERCADE games system. Collisions were also used for hit detection. I also allowed the functionality of different menus, which includes displaying previous scores and working buttons.

Foliage Generator

This tool is intended to allow the user to spawn in different types of foliage; Trees, Flowers and Grass. These are all prefabs which the user can change if needed.

Solo Project Unity C# Tools
MY ROLE

I designed and created this tool with the intention of making terrain generation easier for designers during game production. This tool has a range of functionality and is also designed using stylesheets. Spawning Radius, Spawning Density and Scaling (Min/Max Size) are all used to help the user spawn in different prefabs. I also designed the functionality of the Tool Menu, which includes resizing and docking the window so designers could edit the settings of the tool whilst actively looking at the game scene. Planning using sprints was used for this project to help me keep on track with the productivity of this tool.

OPENGL PROJECT

This OpenGL project was created to understand the functionality of using graphics libraries. I was tasked to create multiple cubes which all had textures which were loaded in via a file. Lighting was also used in this project to further develop my knowledge and understanding in this project. (Ambient, Diffuse, Specular) Shininess was also present.

SNAGGLE

Snaggle is a dungeon crawler which is more of a looter-shooter as the player must traverse dungeon rooms which has real-time enemy spawning and drop chances. The player must find keys to progress through different areas within the dungeon.

Solo Project Unity C# AI
MY ROLE

I had the responsibility for creating the map using tilemaps to create an immersive experience for the player. I was also in control of creating the mechanics of the game which would include Enemy Spawning, NPC's/Shop and Player/Enemy Movement. I also programmed the ability to have a chance at dropping specific loot which would allow the player to seek other reasons to defeat enemies. I also designed the functionality of the animations, which include animator and its node system to blend between different animations.

SOULBOUND

Soulbound is a top-down puzzle adventure focused on environmental storytelling and assisted player exploration. The player guides a lost soul through environmental obstacles which help uncover narrative elements through gameplay.

Collaborative: [1/2] Unity C# AI
MY ROLE

I was responsible for designing and implementing core gameplay systems, such as point and click interactions with visual aids (shaders). Puzzle interactions like, moving obstacles and pipe connecting. AI behaviours, which include; path finding (wandering). I also designed the functionality of the Main Menu, which includes animations and working buttons which could load and exit the game. I worked closely with the level designer/modeller to ensure assets were integrated into functional gameplay.

CATSTRONAUT

Catstronaut is a third person creature-capture adventure where players explore an alien planet filled with interactive wildlife. The player must discover different species of creatures and capture them accordingly before the planet gets destroyed.

Collaborative: [1/24] Unreal Engine C++ Blueprints
MY ROLE

I implemented core player abilities, enemy spawning and interaction systems using C++ and Blueprints. I focused on making sure systems were developed with the intentions of making it designer-friendly, enabling rapid interation during development. I was appointed QA (Quality Assurance) making sure that any bugs were detected and fixed, creating the appropriate documentation to support this. I worked closely with other Technical Designers and Game Designers to ensure worflow was developed efficiently.
